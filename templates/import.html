Import Aeries Grades

How to Import:

  1. Go to Aeries and navigate to your Gradebook Details page
  2. In the assignments table, select from the column headers (Description, Category, Score...) down to the category totals at the bottom
  3. Copy the selection (Ctrl+A to select all, then Ctrl+C)
  4. Paste it into the text box below (Ctrl+V)
  5. Enter your class name and teacher name
  6. Click "Import Grades"

Note: Category weights will be set to equal distribution by default. You can adjust them later if needed.
